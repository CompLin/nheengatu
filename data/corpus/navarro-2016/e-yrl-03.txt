se igara
# minha canoa

ne manha
# tua mãe

i pindá-itá
# os anzóis dele

yané tawa
# nossa cidade

pe ruka
# casa de vocês

aintá xirura-itá
# calças deles

Aé usú se irũmu.
# Ele vai comigo.

Maria upitá ne ruakí.
# Maria fica perto de ti.

Pedro usú apekatú pe suí.
# Pedro vai longe de vocês.

Reruri timbiú ixé arama.
# Traga comida para mim.

Amunhã pindá-itá indé arama.
# Faço anzóis para você.

Apurakí indé arama.
# Trabalho para ti.

Reruri aé ixé arama.
# Traga-o para mim.

Apurakí Maria supé.
# Trabalho para Maria.

Anheẽ “puranga ara” aintá supé.
# Digo “bom dia” para eles.

Unheẽ paá yepé ara i apigawa-itá supé...
# Contam que disse um dia aos seus homens...

Ixé apiripana xirura i xupé.
# Eu compro calça para ele.

Maria usú se rapé rupí.
# Maria vai pelo meu caminho.

Kuíri penhẽ pemaramunhã yané rupí.
# Agora vocês brigam por nós.

Renheẽ kariwa nheenga rupí.
# Fale em língua de brancos.

Muíri rupí taá kwá kamixá?
# Por quanto é esta camisa?

Ayúri se rupí.
# Vim por mim.

Remunhã sangawa rupí.
# Faça de acordo com o modelo.

Kwá rupí aikwé siía mirá.
# Por aqui há muitas árvores. 

Amú pituna piterarupí, asendú yepé nheenga.
# Pelo meio da outra noite, ouvi uma voz.

Aintá usu nhaã tatatingawasú piterarupí...
# Elas foram pelo meio daquela fumaça...

Aintá umuyeréu mirá-itawasú aintá ararupí.
# Eles rolaram grandes paus por cima deles.

Iwí wirarupí paá aikwé pé...
# Contam que, por baixo da terra, havia caminho... 

Se riré usika ana amú apigawa.
# Depois de mim chegou outro homem.

Murasí riré, aé ukiri ana retana.
# Após o baile, ele dormiu muito.

Apurandú ne suí maã aputari akwáu.
# Pergunto de você o que quero saber.

Indé resemu se suí.
# Você nasceu (ou saiu) de mim.

Ixé ayúri paranã suí.
# Eu venho do rio.

Aé uyupirú ana uwatá garapá suí.
# Ele começou a andar desde o porto.

Ixé asarú indé amukwesé suí.
# Eu te espero desde anteontem.

Ixé asarú indé amukwesé suiwara.
# Eu te espero desde anteontem.

se pu
# minha mão

ne mimbira
# teu filho

tukana tĩ
# o bico do tucano

ne tawa
# tua cidade

se igara
# minha canoa

yané putira
# nossa flor

Aikwé kamixá-itá puranga nhaã piripanasawa ruka upé.
# Há camisas bonitas naquela loja.

Aikwé mukũi igara paranã upé.
# Há duas canoas no rio.

Aikwé raẽ será tuxawa tuyuwé?
# Existe ainda o velho tuxaua?

Aiwã amana.
# Já haverá chuva.

Aiwã pituna.
# Logo vem a noite.

Aiwã tẽ kurí.
# Logo será (ou daqui a pouco).

Xukúi kamixá.
# Aqui está a camisa.

Xukúi sekuyara.
# Aqui está o dinheiro.

Kwá garapá mirĩ, nhaã umbaá.
# Este porto é pequeno, aquele não.

Kwá tawa puranga, nhaã tawa puxiwera.
# Esta cidade é bonita, aquela cidade é feia.

Kwá-itá mirá santá, nhaã-itá membeka.
# Estas madeiras são duras, aquelas são moles.

Amaã-putari kwá uri waá-itá.
# Quero ver estes que vêm.

Asú kurí São Gabriel kití.
# Irei a São Gabriel.

Wirandé kurí yamunhã yané ruka.
# Amanhã faremos nossa casa.

Repitá kurí wirandé iké?
# Você vai ficar amanhã aqui?

wirandé kurí tenhẽ
# amanhã mesmo

Mairamé taá kurí indé repurungitá nheengatú?
# Quando você falará nheengatu?

Se mena usika kurí paranã suí.
# Meu marido vai chegar do rio.

Mairamé kurí bũa, ixé asú kurí ayuká indé arama kwá tukunaré.
# Quando eu for grande, eu vou matar para você este tucunaré.
