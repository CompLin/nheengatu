Murasí iwaka upé
# Baile no céu

Amú pituna ramé, Maria umbeú ana amú marandua i mimbira supé:
# Na outra noite, Maria contou outra história a seu filho:

“Yepé ara, paá, São Pedro umunhã yepé murasí iwaka kití.
# “Dizem que, um dia, São Pedro fez um baile lá para os lados do céu.

Ape, usenũi panhẽsuú usú arã umaã murasí.
# Então chamou todos os animais para irem ver o baile.

Ape yautí, paá, unheẽ:
# Contam que, então, o jabuti disse:

— Ti mayé asú akití.
# — Não há como eu ir para lá.

Ixé amaã arama yepé nhaã murasí, ma nti arikú se pepú awewé arã.
# Era para eu ver aquele baile, mas não tenho (minhas) asas para voar.

Ape té, paá, usasá sumuara urubú.
# Contam que, nesse momento mesmo, passou o seu amigo urubu.

— Eh compadre! unheẽ, paá, indé nti será rerasú ixé iwaka kití amaã arã murasí?
# Contam que disse: — Eh compadre, você não me leva para o céu para ver o baile?

— Ah!, paá, unheẽ, anhũresú kwá ararapewa kwara upé.
# Contam que disse: — Ah! só vai (se for) dentro deste violão.

— Eré!
# — Certo!

Ape yautí uyuruari ararapewa kwara upé.
# Então o jabuti embarcou dentro do violão.

Usú ana.
# Foram.

Ape, paá, urubú uwewé i irũmu.
# Contam que, então, o urubu voou com ele.

Usú té iwaka kití.
# Foram até o céu.

Iwaka-pe murasí surí.
# No céu o baile foi alegre.

Pisayé ramé upawa ana murasí.
# À meia-noite acabou o baile.

Panhẽsuú usú ana iwí kití, aintá uyuíri panhẽ, aintá uwiyé panhẽ.
# Todos os animais foram para a terra, eles voltaram todos, eles desceram todos.

Ma yautí nti urikú i pepú uwewé arama.
# Mas o jabuti não tinha (suas) asas para voar.

Aintá resarái i suí.
# Esqueceram-se dele.

Upitá ape.
# Ficou ali.

Ape i kwema ara.
# Então amanheceu o dia.

Uyeréu, umaã iwí kití masuí uri iwaté kití.
# Virou, olhou para a terra donde veio para cima.

Aé nti rẽukwáu mayé umunhã uyuíri arama.
# Ele ainda não sabia como fazer para voltar.

Uyeréu, uwari asuí, poh!
# Virou, caiu dali, poh!

Iwí kití uwari, poh! Iwí kití, kwá kití, makití yaikú.
# Caiu para a terra, poh! Para a terra, para cá, para onde estamos.

Ape yautí kwera upupuka pá.
# Então o jabuti (que “já era”) rebentou todo.

Yawerã, paá, perewa rangawa nhuntu nhaã i pirera.”
# Dizem que é por isso que são somente cicatrizes (sinais de feridas) aquele seu casco.”
