Akayú pisasú usika.
# Chega o ano novo.

Akayú pisasú usika uikú.
# O ano novo está chegando.

São Gabriel upé mira-itá uyumuatiri usarú arama pisayé.
# Em São Gabriel as pessoas se ajuntam para esperar a meia-noite.

Maria usenũi ana mira-itá upinaitikawera sendawa ruakí uri arama suka kití.
# Maria convidou pessoas que costumam pescar perto de sua comunidade para virem a sua casa.

Uri apigawa-itá unheengariwera umusurí arama murasí.
# Vêm homens cantadores para alegrar a festa.

Pituna sakurana.
# A noite está pouco quente (morna).

Panhẽ mira usú ukara kití.
# Todas as pessoas vão para fora.

Kuíri supí aintá uyumusurí-kwáu.
# Agora de fato eles podem se divertir.

Maria usenũi aintá:
# Maria os chama:

Yasú panhẽuka kwara kití!
# Vamos todos para dentro de casa!

Aikwé bũa timbiú penhẽ arama!
# Há muita comida para vocês!

Aikwé bũa meyú-itá iurá árupi pembaú arama!
# Há grandes bijus sobre o jirau para vocês comerem! 

Antônio, Maria mimbira, uyupirú unheengari.
# Antônio, filho de Maria, começa a cantar.

Aintá unheẽ i xupé:
# Dizem a ele:

Indé nheengarisararana.
# Você é um mau cantor. 

Repituú renheengari!
# Pare de cantar!

Maria unheẽ:
# Maria diz:

Rute, indé renheengari ramé maã, panhē mira upitá maã suri.
# Rute, se você cantasse, toda a gente ficaria feliz.

Panhẽ mira umbaú meyú.
# Todas as pessoas comem beiju.

Yepé apigawa kawera usika murasí upé.
# Um homem bêbado chega à festa.

Maria uyakáu aé:
# Maria o repreende:

Remaã mamentu reikú!
# Veja só onde você está!

Indé nti rerikú tĩsawa?
# Você não tem vergonha?

Reikuntu mimi!
# Fique quieto ali!

Resú mimi kití!
# Vá para lá!

Nhaã kawera uyana tenhuntu usikari arama i igara:
# Aquele bêbado correu em vão procurando sua canoa:

aé nti ana uwasemu.
# ele já não a encontrou.

Maria unheẽ:
# Maria diz:

Kawera-itá yawé tẽ.
# Bêbados são assim mesmo.

Ne mairamé aintá rurí.
# Nunca são felizes.
