Xukúi se pururé.
# Aqui está minha enxada.

Nti akwáu mamé uikú kwá kunhã yara.
# Não sei onde está a desta mulher.

Se mũ pirasua, ne yara umbaá.
# Meu irmão é pobre, o teu não.

Nhaã uka se yara.
# Aquela casa é minha.

Pedro ruka mirĩ. Pe yara turusú.
# A casa de Pedro é pequena. A de vocês é grande.

Kwá ne igara.
# Esta é tua canoa. 

Awá yara taá nhaã?
# Aquela é propriedade de quem?

Awá igara taá nhaã?
# Canoa de quem é aquela?

Se manha puranga piri ne yara suí.
# Minha mãe é mais bonita que a sua.

Kwá apigawa pirasua piri nhaã suí.
# Este homem é mais pobre que aquele.

Kwá kunhã unheengari puranga.
# Esta mulher canta bem.

Kwá kunhã unheengari puranga piri nhaã suí.
# Esta mulher canta melhor que aquela.

Se manha puranga mayé ne yara yawé.
# Minha mãe é tão bela como a sua.

Se manha puranga mayé ne yawé.
# Minha mãe é tão bonita como tu.

Kwá apigawa pirasua mayé nhaã yawé.
# Este homem é tão pobre como aquele.

Kwá kunhã unheengari puranga mayé nhaã yawé.
# Esta mulher canta tão bem como aquela.

Aé urikú suaya pirá yawé.
# Ele tem cauda como peixe.

Yané paya-itá ikewara.
# Nossos pais são daqui.

Kuximawara-itá upurakí retana.
# Os antigos trabalhavam muito.

Uka mirá suiwara.
# A casa é de madeira.

Karuka rirewara pituna.
# O que é após a tarde é a noite.

Maria upurandú Pedro resewara.
# Maria perguntou a história de Pedro.

Maria upurandú sesewara.
# Maria perguntou a história dele.

Asú amupinima se iwí resewara.
# Vou escrever o que é relativo à minha terra.

Asú amupinima sesewara.
# Vou escrever o que é relativo a ela.

Aé upuú siía kumandamirĩ.
# Ele colhe muito feijão.

Aé umunhã siía makira.
# Ele faz muitas redes.

Aé umunuka kwaíra mirá.
# Ele corta poucas árvores.

Aé umururú kwaíra iwí.
# Ele rega pouca terra.

Remeẽ ixé arama uí mirĩ nhuntu.
# Dê-me só um pouco de farinha.

Kwaíra nhuntu, paá, aé uú kaxirí.
# Contam que ele bebeu caxiri um pouco, somente.

Sapukaya-apigawa unheengari retana.
# O galo canta muito.

Aé upitá piaíwa reté ne irũmu.
# Ele fica muito bravo com você.

Maria upurakí xinga.
# Maria trabalha pouco.

Aé tepusimanha xinga.
# Ele está um pouco sonolento.

Aé upurungitá mirĩ.
# Ele fala pouco.

Senundé kití xinga aé usuantí yepé tatú.
# Um pouco à frente dele, ele encontrou um tatu.

Mairamé usika, apekatú xinga upitá.
# Quando chegou, ficou um pouco longe.

Maãyawé nhaá kurumĩwasú payé xinga, aé umaã, paá, i anga rupí maã kunhãmukú-itá umunhã i xupé.
# Como aquele rapaz era um pouco pajé, contam que ele viu, por sua sombra, o que as moças faziam para ele.
