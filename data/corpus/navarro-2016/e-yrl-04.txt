Se paya, paá, usú ana kaá kití, ma aé nti usú.
# Dizem que meu pai foi para a mata, mas ele não foi.

Apigawa upurungitá waá se paya, ma aé ukiri waá se tutira.
# O homem que fala é meu pai, mas o que dorme é meu tio.

Kunhã-itá indé remaã waá ne mimbira.
# As mulheres que você vê são suas filhas.

Apigawa i paya waá umurari iké, sera José.
# O homem que o pai dele mora aqui, seu nome é José.

Pirá sera waá pirarukú, turusú retana.
# O peixe cujo nome é pirarucu, é muito grande.

Amaã ana yepé kurumĩ, kunhã puranga waá mimbira.
# Vi um menino, filho da mulher que é bonita.

Kunhã i mimbira waá indé remaã, usú paranã kití.
# A mulher cuja filha você vê, vai ao rio.

Aikwé yepé uka turusú waá.
# Há uma casa que é grande.

Uií ara umanũ waá-itá ara.
# O dia de hoje é o dia dos que morreram.

Akwáu amunhã waá.
# Sei o que faço.

Akwáu maã amunhã waá.
# Sei o que faço.

Aé upitá-putari mamé puranga waá.
# Ele quer ficar onde é bonito.

Maria rendawa
# comunidade de Maria

se rendawa
# minha comunidade

ne rendawa
# tua comunidade

yané rendawa
# nossa comunidade

pe rendawa
# a comunidade de vocês

aintá rendawa
# a comunidade deles

sendawa
# comunidade dele
# comunidade dela

Uka puranga.
# A casa é bonita.

uka puranga
# casa bonita

Se ruka puranga.
# Minha casa é bonita.

Suka puranga.
# A casa dele é bonita.

Aintá ruka puranga.
# A casa deles é bonita.

Amaã pé kaá pitérupi.
# Vejo o caminho no meio do mato.

Ne rapé i pukú.
# Teu caminho é longo.

Atuká ukena.
# Bati à porta.

Atuká pe rukena.
# Bati à porta de vocês.

se pú
# minha mão

ne manha
# tua mãe

igara mirá suiwara
# canoa de madeira

kisé itaité suiwara
# faca de aço

Ixé São Paulo suiwara.
# Eu sou de São Paulo.

Ixé aikú iké kwesé suiwara.
# Eu estou aqui desde ontem.

Aikú nhaã tawa ruakí.
# Estou perto daquela cidade.

Aikú suakí.
# Estou perto dela.

Kwá mirá turusú.
# Esta árvore é grande.

Arikú yepé uka turusú.
# Tenho uma casa grande.

Awá urikú turusú iwí umeẽ-kwáu awá nti urikú supé.
# Quem tem muita terra pode dar para quem não tem.

igara mirĩ
# canoa pequena

itá mirĩ
# pedra pequena

Kwaíra ramé rẽ, paá, ixé, purangamirĩ aikú.
# Dizem que, quando eu ainda era pequeno, bonitinho estava.

Aé kwera, taité!
# Ele já era, coitado!

Pedro uyuká ana yawareté. Aé kwera.
# Pedro matou a onça. Ela já era.

Awá kwera?
# Quem era?

Asuiwara kwera, mukũi apigawa nhũ upitá umanhana uikú yané itá uka.
# Desde então, somente dois homens ficaram vigiando nossa casa de pedra.
