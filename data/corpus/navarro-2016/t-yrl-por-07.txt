# Producer: Juliana Lopes Gurgel
# Last update: May 15, 2022

Maria umunhã timbiú.
# Maria faz comida.

Maria umemũi kurí timbiú i anama-itá supé.
# Maria vai cozinhar comida para seus parentes.

Aé umunhã kurí kwaíra uí yuíri.
# Ela fará um pouco de farinha também.

Aé upisika maniaka, upiruka aé, ukitika aé i iwisé irũmu.
# Ela pega a mandioca, descasca-a, rala-a com seu ralador.

Ariré, aé upurakari tipití maniaka kitika irũmu.
# Depois, ela enche o tipití com a mandioca ralada.

Utipika aé, umusemu manikwera, umburi pá urupema upé, umuáu uí, umutiní aé yapuna upé.
# Espreme-a, faz sair a manipueira, põe tudo na peneira, peneira a farinha, torra-a no forno.

Aé upisika mukũi sapukaya, uyuká aintá i pu irũmu umemũi arama aintá.
# Ela pegou duas galinhas, matou-as com suas mãos para cozinhá-las.

Aé umusaimbé i kisé, umunuka musapiri pirá i mena ururi waá paranã suí.
# Ela afia sua faca, corta três peixes que seu marido trouxe do rio.

Aé unheẽ:
# Ela diz:

Se mimbira, reyuri repitimũ ixé se purakisawa upé.
# Meu filho, venha ajudar-me no meu trabalho.

Maã taá reputari ixé amunhã?
# Que você quer que eu faça?

Remundeka tatá ixé arama.
# Acenda o fogo para mim.

Eré.
# Certo.

Amundeka kurí tatá indé arama.
# Acenderei o fogo para você.

Ixé aputari amemũi musapiri pirá yambaú arama.
# Eu quero cozinhar três peixes para nós comermos.

Té remburi kiínha pirá resé.
# Não ponha pimenta no peixe.

Ixé ayumusé pirá kiinhaíma.
# Eu gosto de peixe sem pimenta.

Té remundeka tatá iwí upé.
# Não acenda fogo no chão

Remunhã tatá memũitawa upé.
# Faça fogo no fogão.

Tatá i awaité.
# Fogo é perigoso.

Antônio umundeka ana tatá.
# Antônio acendeu o fogo.

Maria umburi kamutí tatá árupi.
# Maria põe a panela de barro sobre o fogo.

Yepeawa upawa ana.
# A lenha acabou.

Antônio ururi ana piri yepeawa memũitawa kití.
# Antônio já trouxe mais lenha para o fogão.

Tik!
# Nossa!

Manha, repuíri timbiú!
# Mãe, mexa a comida!

Aé ukái uikú!
# Ela está-se queimando!

Antônio, indé reputari ixé amunhã meyú yuíri?
# Antônio, você quer que eu faça biju também?

Eẽ, meyú sé.
# Sim, biju é gostoso.

Ariré, Maria usenũi i anama-itá:
# Depois, Maria chama seus familiares:

Yasú yambaú!
# Vamos comer!
