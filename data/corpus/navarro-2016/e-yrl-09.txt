Ape té usasá sumuara urubú.
# Nesse momento mesmo passou seu amigo urubu.

Wirandé tẽ kurí amunhã se ruka.
# Amanhã mesmo farei minha casa.

Wirandé tenhẽ kurí amunhã se ruka.
# Amanhã mesmo farei minha casa.

Ixé té amunhã ana timbiú.
# Eu mesmo fiz a comida.

Yawé tẽ aputari.
# É assim mesmo que eu quero.

Aintá umupuranga tupauku.
# Enfeitaram a igreja.

Tá uyuká yepé tatú paranã upé.
# Mataram um tatu no rio.

Yepé ara, paá, São Pedro umunhã murasí iwaka kití.
# Dizem que, um dia, São Pedro fez um baile lá para o céu.

Mayé taá asú awatá se retama kití?
# Como vou andar lá para a minha terra?

Mamé taá uikú ne retama? Pará kití.
# Onde está tua terra? Para os lados do Pará.

yasí pitera kití
# lá para o meio do mês

Iwakape murasí surí.
# No céu o baile foi alegre.

Yepé uwapika igara gantĩme, amú uwapika yakumãme.
# Um sentou na proa da canoa, outro sentou na popa.

Paranãme igara upitá ana.
# No rio a canoa ficou.

Yautí usú ararapewa kwara upé.
# O jabuti foi dentro do violão.

Usuaxara aé i marika pupé.
# Respondeu ele dentro da barriga dela.

Upitá uka pupé.
# Ficou dentro de casa.

Pedro usemu tupauku kwara suí.
# Pedro saiu de dentro da igreja.

Ixé amaã arama yepé nhaã murasí, ma nti arikú se pepú awewé arã.
# Era para eu ver aquele baile, mas não tenho minhas asas para voar.

Ixé apitá arama yepé iké.
# Era para eu ficar aqui.

Watarampuá, nhaã kurumiwasú kirimbawa piri waá yandé retamawara aintá suí, umendari arama waá yepé xe irũmu, nti rẽ uyana kwá kaxiwera.
# Uatarampuá, aquele moço que é o mais valente dos que são de nossa terra, que era para casar-se comigo, não correu ainda esta cachoeira.

Se kerpi.
# Eu sonho.

Ne kerpi.
# Você sonha.

I kerpi.
# Ele sonha.

Yané kerpi.
# Nós sonhamos.

Pe kerpi.
# Vocês sonham.

Aintá kerpi.
# Eles sonham.

Ixé se akanhemu.
# Eu me assusto.

Indé ne akanhemu.
# Você se assusta.

Aé i akanhemu.
# Ele se assusta.

Yandé yané akanhemu.
# Nós nos assustamos.

Penhẽ pe akanhemu.
# Vocês se assustam.

Aintá tá akanhemu.
# Eles se assustam.

Ixé se resarái.
# Eu me esqueço.

Indé ne resarái.
# Você se esquece.

Aé sesarái.
# Ele se esquece.

Yandé yané resarái.
# Nós nos esquecemos.

Penhẽ pe resarái.
# Vocês se esquecem.

Aintá tá resarái.
# Eles se esquecem.

I kwema.
# Amanhece.

Ee ruí.
# Eu sangro.

Pedro tuí.
# Pedro sangra.

Aintá ruí.
# Eles sangram.
