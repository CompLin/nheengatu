# Producer: Juliana Lopes Gurgel
# Last update: May 15, 2022

São Gabriel upé.
# Em São Gabriel.

Maria usemu igara irũmu.
# Maria sai com a canoa.

Aé usú São Gabriel kití.
# Ela vai a São Gabriel.

São Gabriel tawa puranga, apekatú Barra suí.
# São Gabriel é uma cidade bonita, distante de Manaus.

Siía mira umurari ape.
# Muitas pessoas moram ali.

Aé upiripana kurí maã-itá i mimbira supé, i mena supé yuíri.
# Ela vai comprar coisas para seu filho, para seu marido também.

Aé usika ape, usú yepé piripanasawa ruka kití.
# Ela chega lá, vai a uma loja.

Aé unheẽ yepé meẽsara supé:
# Ela diz a um vendedor:

Ixé aputari yepé kamixá se mimbira supé.
# Eu quero uma camisa para meu filho.

Aé urikú mukũi akayú.
# Ele tem dois anos.

Kwá kamixá puranga retana.
# Esta camisa é muito bonita.

Aé sepiasuíma.
# Ela é barata.

Maã taá aikwé se mena supé?
# Que há para o meu marido?

Maã taá indé reputari ne mena supé?
# Que você quer para seu marido?

Se mena upurakí kurí garapá upé.
# Meu marido vai trabalhar no porto.

Aé uputari yepé xirura pisasú.
# Ele quer uma calça nova.

I xirura-itá suruka.
# As calças dele estão rasgadas.

Kwá puranga.
# Esta é bonita.

Muíri rupí taá kwá xirura?
# Por quanto é esta calça?

Mukũi real.
# Dois reais.

Muíri rupí nhaã?
# Por quanto é aquela?

Nhaã xirura mukũi real yuíri.
# Aquela calça é dois reais também.

Aputari nhaã.
# Quero aquela.

Xukúi xirura.
# Aqui está a calça.

Nti reputari ne maã indé arama?
# Não quer nada para você?

Umbaá.
# Não.

Ariré kurí apiripana maã-itá ixé arama.
# Depois comprarei coisas para mim.

Xukúi sekuyara.
# Aqui está o dinheiro.

Maria usepimeẽ.
# Maria paga.

Meẽsara upupeka panhẽ maã-itá.
# O vendedor embrulha todas as coisas.

Xukúi sekuyaramirĩ.
# Aqui está o troco.

Maria umukwekatu:
# Maria agradece:

Kwekatú reté!
# Muito obrigada!

Ariré, aé usú amú piripanasawa ruka kití.
# Depois, ela vai a uma outra loja.
