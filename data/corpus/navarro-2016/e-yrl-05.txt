Asaisú se manha.
# Amo minha mãe.

Asaisú se maã-itá.
# Sovino minhas coisas.

Yasú tendawa kití yamunhã arama kupixawa.
# Vamos à comunidade para fazer a roça.

Apitá ne ruka upé apurungitá arama ne irũmu.
# Fico na sua casa para falar com você.

Xukúi pirá indé rembaú arã.
# Eis o peixe para você comer.

Remunhã ne kakurí nti arã rewatá remundá se kakurí.
# Faça o seu cacuri para não andar furtando o meu cacuri.

Maria umaã ixé.
# Maria me vê.

Ixé aruri aé kwá kití.
# Eu o trago para cá.

Pedro umaã kurí indé suka upé.
# Pedro vai ver-te na casa dele.

Mundawasú upisika ana aé suka upé.
# O ladrão o apanhou na casa dele.

Asarú remanũ ambaú arã indé.
# Espero você morrer para comê-lo. 

Asú ramé, arasú se mimbira.
# Quando vou, levo meu filho.

Arasú ramé timbiú, nti apitá yumasisawa irũmu.
# Quando levo comida, não fico com fome.

Nti ramé repurakí, indé repitá pirasua.
# Se não trabalhas, tu ficas pobre.

Nti ramé manungara rerikú, remeẽ uí nhuntu.
# Se não tens nada, dá só farinha. 

Kwaíra ramé rẽ ixé, anheengari puranga.
# Quando eu (era) ainda pequeno, cantava bem.

Indé ramé, nti aputari.
# Se for você, não quero.

Asú murakipí ramé.
# Vou na segunda-feira.

Akiri pituna ramé.
# Durmo de noite.

Ayuíri mituú ramé.
# Volto no domingo.

Amú ara ramé ana, paá, aintá upurandú:
# No outro dia, conta-se que eles perguntaram:

Nti aé uputari será indé?
# Ele não quis você?

Mairamé arasú timbiú, ixé nti apitá yumasisawa irũmu.
# Quando levo comida, eu não fico com fome.

Maria usú ana suka kití.
# Maria foi para sua casa.

Ixé apitá ana se retama upé.
# Eu fiquei na minha terra.

Ixé apurungitá ana aikú mairamé indé rewiké.
# Eu estava falando quando você entrou.

Aé nti uyukataka; umaã yepé iwaka kití, unheengari uikú yepé nheengarisawa surí.
# Ele não se mexeu; olhava para o céu, estando a cantar uma canção alegre.

Mayawé ixé maraari aikú yepé, ayenũ se mimbira ruakí, akiri ana.
# Como eu estava cansado, deitei-me perto de meu filho e dormi.

Se manha unheẽ kwera ixé arama:
# Minha mãe dizia para mim:

Nhaã se kurumĩ. 
# Aquele é meu menino.

Kuxiíma, reyúri ramé kwá kití, mayé taá reyúri?
# Antigamente, quando você veio para cá, como você veio? 

Kunhã surí.
# A mulher (ela) é feliz. 

Ixé se rakú aikú.
# Eu estou quente.

Uka sakú.
# A casa é quente.

Aé sakú.
# Ela é quente.

Aintá rakú.
# Elas são quentes.

Se rikwé aikú.
# Eu estou vivo.

Kwá wirá sikwé uikú.
# Este pássaro está vivo.

Maria uyupirú umunhã timbiú i anama-itá supé.
# Maria começa a fazer comida para seus familiares.

Aputari apurungitá ne irũmu.
# Quero falar com você.

Pedro usú upurungitá Maria irũmu suka upé.
# Pedro vai falar com Maria na sua casa.

Ixé axari usasá pituna.
# Eu deixo passar a noite.

Maria upiripana-putari kamixá-itá i mena supé.
# Maria quer comprar camisas para seu marido.

Maria mimbira upurungitá-kwáu nheengatú.
# O filho de Maria sabe falar nheengatu.

Amunhã-pawa se purakisawa.
# Acabei de fazer meu trabalho.

Usika-pawa.
# Acabou de chegar.

Uyenũ-yuíri.
# Voltou a deitar.

Aputari Pedro usika uií.
# Quero que Pedro chegue hoje.

ixé ayúri.
# eu venho.

indé reyúri.
# você vem.

aé uri.
# ele vem.

aintá uri.
# eles vêm.
