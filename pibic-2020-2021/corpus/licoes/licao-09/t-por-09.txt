Baile no céu

Na outra noite, Maria contou outra história a seu filho:

Dizem que, um dia, São Pedro fez um baile lá para os lados do céu.

Então chamou todos os animais para irem ver o baile.

Contam que, então, o jabuti disse:

Não há como eu ir para lá.

Era para eu ver aquele baile, mas não tenho (minhas) asas para voar.

Contam que, nesse momento mesmo, passou o seu amigo urubu.

Contam que disse: Eh compadre, você não me leva para o céu para ver o baile?

Contam que disse: Ah! só vai (se for) dentro deste violão.

Certo!

Então o jabuti embarcou dentro do violão.

Foram.

Contam que, então, o urubu voou com ele.

Foram até o céu.

No céu o baile foi alegre.

À meia-noite acabou o baile.

Todos os animais foram para a terra, eles voltaram todos, eles desceram todos.

Mas o jabuti não tinha (suas) asas para voar.

Esqueceram-se dele.

Ficou ali.

Então amanheceu o dia.

Virou, olhou para a terra donde veio para cima.

Ele ainda não sabia como fazer para voltar.

Virou, caiu dali, poh!

Caiu para a terra, poh!

Para a terra, para cá, para onde estamos.

Então o jabuti (que "já era") rebentou todo.

Dizem que é por isso que são somente cicatrizes (lit. sinais de feridas) aquele seu casco.
